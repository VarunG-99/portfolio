SAR Imaging and Classification using Sentinel-1 Data

Duration: March 2025 – April 2025

Tools: ESA SNAP Toolbox, Sentinel-1 Data, Python

I worked on processing Synthetic Aperture Radar (SAR) imagery from the Sentinel-1 satellite to identify and classify land features using both unsupervised and semi-supervised classification methods. The goal was to explore automated terrain recognition and environmental monitoring using microwave remote sensing techniques.

Key highlights:

  • Preprocessed SAR GRD data (calibration, speckle filtering, terrain correction).
  • Applied k-means clustering and ISODATA for unsupervised classification.
  • Evaluated backscatter intensities to analyze urban vs. vegetative land cover.
  • Gained hands-on experience with the ESA SNAP Toolbox, and interpreted band combinations and histograms for land-type differentiation.

This project strengthened my understanding of radar imaging, signal processing, and real-world applications in geospatial analytics.
